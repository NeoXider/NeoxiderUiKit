# Пример сгенерированного интерфейса (MY_GAME_DESIGN)

Снимок готового результата UI Kit по демо-дизайну: сгенерированные вьюшки
(`Generated/*.g.cs`, `UiIds.g.cs`), заполненный `UiKitConfig.asset`, шпаргалка
`UiKitApi.md`, проектный override-USS и **пользовательские partial-классы**
(`Views/*.cs`, включая `AudioToggleWiring.cs` — обвязка тумблеров звука/музыки под
разметку MY_GAME_DESIGN) плюс адаптер-заготовка NeoxiderTools.

Это референс: так выглядит проект после нажатия «Сгенерировать вьюшки» и
«Создать объекты на сцене» и подключения кастомной логики. Файлы `.g.cs`
перегенерируются, `Views/*.cs` — пользовательские (не затираются).
